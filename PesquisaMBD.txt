No SQL Server 2014 (12.x) e anteriores, todas as versões davam suporte a 900 bytes para todos os tipos de índice.	O número máximo de bytes em qualquer chave de índice clusterizado não pode exceder 900. Para uma chave de índice não clusterizado, o máximo é 1.700 bytes.
